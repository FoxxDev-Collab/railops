Track Rendering Style

What should the yard schematic look like? This sets the visual identity of the whole feature.

A — Blueprint / Engineering

Dark background, cyan lines, monospace labels, grid overlay. Classic technical drawing feel. Tie marks on rails. Clean and authoritative.

B — Clean Schematic / Transit Map

Light background, bold rounded lines, color-coded by track type, pill-shaped capacity badges. Modern and approachable, like a transit diagram.

C — Realistic / Model Railroad

Dual rails, ballast texture, wooden ties, realistic bumpers. Feels like looking at a model railroad from above. More visual detail, more immersive.
