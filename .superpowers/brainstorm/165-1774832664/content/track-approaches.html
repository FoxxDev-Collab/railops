Track Layout Approach

How should users build their yard layouts? Pick the approach that feels right.

A — Pre-built Yard Templates

Users pick from templates (small yard, classification yard, staging, interchange) and customize track names, counts, capacity, and types. No drawing — just configure.

Fastest to ship. Clean output guaranteed. Limited flexibility.

B — Snap-Together Track Pieces

Drag pre-made pieces (straight, curve, turnout, bumper) from a palette onto a grid. Pieces snap together at endpoints. Build any yard shape from components.

More flexible than templates. Still constrained enough to look clean. Medium complexity.

C — Templates + Customization

Start from a template, then add/remove individual tracks, rename them, adjust capacity, and reorder. The layout auto-renders as a clean schematic. Best of both worlds.

Fast start with templates. Still flexible. Output always looks professional.
