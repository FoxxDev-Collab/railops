How should the three map modes be organized?

The map page needs three views: Location Overview (existing), Track Sections with waypoints, and Yard Details. How should users switch between them?

A

Top Tab Bar

Three tabs across the top of the map area: Locations · Track Layout · Yard Detail. Clean separation, always visible. Yard Detail tab is contextual — only active when a location is selected.

Segmented Toggle in Toolbar

Unified Canvas with Layer Toggle
