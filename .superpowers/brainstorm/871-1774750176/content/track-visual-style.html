Track Visual Style

Which railroad map style fits best? Think RR Tycoon / train map aesthetics.

A — Vintage Railroad Map

Warm parchment tones, visible crossties on mainline, brown palette. Classic RR Tycoon feel. Locations as solid station dots with labels.

B — Modern Schematic

Dark background, clean bold lines, transit-map clarity. Mainline is white, branches are colored. Matches the app's dark mode well.

C — RR Tycoon Game Style

Dark background with warm accents. Double-rail rendering with wooden ties, station buildings as icons, game-inspired palette. Waypoints as small diamonds.

D — Survey / Topo Map

Light background, USGS-style railroad symbols (line with perpendicular ticks), serif italic labels. Classic cartographic look. Clean and authoritative.
