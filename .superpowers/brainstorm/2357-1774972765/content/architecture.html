Key Decisions

Extraction

Node.js script clusters SVG paths by spatial proximity, extracts each vehicle as a standalone SVG with tight viewBox. Light + dark mode variants.

Storage

Static files in public/silhouettes/ for fast serving. DB table for metadata, selection UI, and future user uploads.

Dark Mode

Both light and dark SVG variants extracted. Component swaps based on current theme using next-themes.

Consist View

Replace generic RollingStockIcon shapes with actual assigned silhouettes. Fallback to generic if none assigned.
