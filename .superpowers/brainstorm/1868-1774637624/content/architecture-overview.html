Layout Editor — Architecture Overview

Three-layer system: Editor → Detail Views → Session Display

System Architecture
Layer 1: Railroad Overview Editor
React Flow canvas
• Location nodes (drag, place, snap-to-grid)
• Track edges (click-to-connect, magnetic snap)
• Toolbar: add location, draw track, zoom controls
• Minimap + pan/zoom
• Custom node types per LocationType
Data: LayoutCanvas model (node positions, edge paths)
Layer 2: Location Detail View
Konva canvas (react-konva)
• Yard track diagrams (arrival, class, departure)
• Industry spurs and sidings
• Car positions on tracks
• Turnout/switch indicators
• Click-to-zoom from overview
Data: LocationCanvas model (track geometry, car slots)
Layer 3: Session Display
Read-only overlay on same canvases
• Train position markers (animated)
• Active switch list highlights
• Car movement indicators
• Dispatcher click-to-interact
• Crew polling (5-10s refresh)
Data: OperatingSession + SessionTrain state
